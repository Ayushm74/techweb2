# Futuristic site

Website made in react in futuristic style for no existing revolutionary auth technology

## Run Locally

Clone the project

```bash
https://github.com/Ayushm74/techweb1
```

Go to the project directory

```bash
  cd Futuristic-site
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run dev
```
